# GYM App Demo

This is a static demo version of the GYM app. No backend or database is required. All data is hardcoded for demonstration purposes.

## How to Run

1. Install dependencies (only Express is needed for the static server):
   
   npm install express

2. Start the demo server:
   
   npm start

3. Open your browser and go to:
   
   http://localhost:3001

## Features
- Member Management (demo)
- Attendance Tracking (demo)
- Class Scheduling (demo)
- Payment Processing (demo)
- Trainer Dashboard (demo)
- Owner Analytics (demo)

This demo is for showcasing the app UI and features only. No real authentication or data storage is used.
